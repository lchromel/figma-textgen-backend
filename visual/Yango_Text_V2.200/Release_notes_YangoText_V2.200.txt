Basic information
1 Font name: Yango Text
2 TrueType Version record: 2.200
3 Character set: All European Latin, Standard Arabic, Hebrew, Cyrillic
4 Supported scripts: Latin, Arabic, Hebrew, Cyrillic
5 Number of glyphs: 1184
6 Weights/Styles: Regular, Medium, Bold, XBold, Black
7 Usage: Text and Display

Technology
1 File formats: The files have been delivered in the TrueType-based OpenType format, for use in all major Operating Systems, and also as TTF for use in apps, and in web font format: WOFF; WOFF2.
2 Embedding settings: Editing of the document is allowed (editable mode), subsetting is allowed.
3 Screen optimization information: PixelFix CTAA screen optimization technology.
4 Default numerals: Tabular. Proportional figures accessible via OpenType features.
